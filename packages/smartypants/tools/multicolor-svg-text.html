Different Font Examples:

How to use your own TTF fonts:

  1. Google Fonts: Add @import in CSS (easiest)
  2. Local TTF file: Use @font-face with src: url('your-font.ttf')
  3. System fonts: Just use font-family="YourSystemFont"

To use your TTF file:

@font-face {
    font-family: 'MyAwesomeFont';
    src: url('./fonts/my-awesome-font.ttf') format('truetype');
}

/* Then use it: */
font-family="'MyAwesomeFont', Arial, sans-serif"
